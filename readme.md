# 项目描述
项目整体是一个 ROS2 + JAKA 双臂 + AGV + 头部 RGB-D 相机 的工件装配场景。

# 流程说明
1. 双臂运动到安全位置
2. 头部相机拍照定位工件位置和摆放角度
3. 左臂运动到抓取前的过渡位置
4. 左臂打开夹爪
5. 左臂运动到工件位置
6. 左臂关闭夹爪，等待1秒
7. 左臂上抬200mm，然后返回抓取前过渡位置
8. 左臂移动到安装底座上方
9. 左臂向下运动尝试将工件安装入底座，同时开始监听左臂的力控传感器数据，若传感器数据超阈值，则终止安装动作，上抬20mm后，RZ顺时针旋转5度，再次尝试。循环次操作，直到安装成功
10. 左臂撤离工件，并回到安全位置
11. 右臂随后来到工件位置，取回工件，并放置到另一侧
12.右臂回到安全位置
